使用简要说明

1、主文件：MaterThesis.tex
2、配置文件：CASThesis.cls、CASthesis.cfg
3、各章节文件位于chapter下
4、图片资源位于figures下


运行方式：

执行主文件即可
编译方式：XeLaTex

若参考文献选择源是bib，则需要先编译Bibtex生成bbl文件


注：尽管我对该模板做了很多配置上的调整，但水平有限，目前还不能完全符合学校要求的格式，生成毕业论文过程中需要额外处理的工作如下：
1、英文摘要页和其他页面的页眉使相同的，需要手动替换英文摘要页的页眉，处理方法是分别生成中英文的形式的页眉，然后利用PDF编辑器直接对英文页面进行替换
2、原创声明页面目前的源是图片，配置文件中有非图片的文字源，水平有限没有配置过来，我处理是直接在Word生成原创声明的PDF文件再进行替换（生成过程中注意尺寸）
3、目录页面目前的摘要和目录这个几个都是比较大的，它是和标题的字体一样的，所以在生成目录也时候需要在配置文件中找到设置摘要和目录的字体和大小，然后生成目录页，最后和上面两条一样，利用PDF编辑器进行简单的目录替换就可以了。

Final_MasterThesis文件夹是可以出来后的最终版，里面包含需要替换的页面
